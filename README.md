# HelloWorld Extension

HelloWorld is a simple Visual Studio Code extension that displays a "Hello, World!" message when the command is executed.

## Features

- Displays a "Hello, World!" message in the VS Code editor.
- Provides a simple example of how to create and use a basic VS Code extension.

## Requirements

No special requirements are needed to use this extension.

## How to Use

- Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P` on macOS).
- Type "Hello World" and select the command "HelloWorld: Say Hello".
- A "Hello, World!" message will appear in the bottom right corner of the editor.
